## English Description

In this slide, we can see the **database layer used in our system.**

In any AI or software platform, **data storage and management** are very important components.

To manage and store system data efficiently, we have used the **MySQL database.**

**MySQL** is used to store the main system data such as:

- User information  
- System records  
- Data generated by different modules  

In addition, we also use a **CSV-based data storage system.**

**CSV files** are mainly used for:

- Data analysis  
- Reporting  
- Storing research-related information  

By combining **MySQL and CSV storage**, the system can efficiently **store, process, and analyze large amounts of data.**
