## English Description

In this slide, we can see the **frontend technology stack used in our system.**

The frontend is the part of the system with which the **user directly interacts.**

Our AI platform interface is built using **modern web technologies.**

First, we used **HTML5**, which provides the **basic structure and layout** of the web application.

Second, we used **CSS3**, which is responsible for **styling and designing the user interface** to make it visually attractive and user-friendly.

Finally, **JavaScript** is used to add **interactivity and dynamic functionality** to the system.

By combining these technologies, we have created a **responsive and interactive web interface** that allows users to access different modules and features of the AI platform.
