## English Description

In this slide, we can see the **backend technology stack used in our system.**

The backend is responsible for handling the **core logic, processing, and functionality** of the system.

It manages **data processing, AI operations, and communication between different components** of the platform.

In our project, we primarily used **PHP and Python** for backend development.

**PHP** is used to manage **web server operations and handle communication between the frontend interface and the backend system.**

Additionally, **Python** is used for **AI processing, data analysis, and cybersecurity-related operations.**

The system also integrates **APIs**, which allow different services and modules to **communicate and exchange data efficiently.**

By combining these technologies, the system provides a **powerful and scalable backend engine** that supports the overall functionality of the AI platform.
